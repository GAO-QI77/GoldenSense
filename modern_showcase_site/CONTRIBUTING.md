We love your help! To keep the project clean and consistent, please follow these guidelines.

## How to Contribute
1. **Fork** the repository on GitHub.
2. **Clone** your fork to your local machine.
3. **Create a branch** for your feature or fix: `git checkout -b feat/my-new-feature`.
4. **Commit** your changes with clear messages.
5. **Push** to your fork.
6. **Open a Pull Request** (PR) to the `main` branch.

## Code Standards
- Use **Prettier** for code formatting.
- Follow **React Functional Components** patterns.
- Ensure all new components are **responsive**.
- Document any new complex logic or utility functions.

## Community Guidelines
- Be respectful and professional.
- Use the issue tracker for bugs and feature requests.
- Provide clear steps to reproduce bugs.

By contributing, you agree that your contributions will be licensed under the project's MIT License.
